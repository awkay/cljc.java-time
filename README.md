# cljc.java-time

A Clojure(Script) library which provides the java.time api through kebab-case-named function vars.

get it from [Clojars](https://clojars.org/cljc.java-time)
 
 In .cljc file
 ```
 (ns my.cljc
   (:require  [cljc.java-time.local-date :as ld])
   
   
   (ld/parse "2019-01-01")
   
 ```
